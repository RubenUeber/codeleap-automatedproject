<h1>E2E Automated Test Suite with Cypress</h1>

<h2>Overview</h2>

<h3>This repository contains an end-to-end (E2E) automated test suite built using Cypress, a modern JavaScript testing framework. The tests are designed to simulate user interactions and verify the functionality of Codeleap project.</h3>

<h2>Prerequisites</h2>

<h3>Before running the tests locally, ensure you have the following installed:

Node.js and npm
Cypress (install via npm: npm install cypress)</h3>

<h2>Running the Tests</h2>

<h3>To run the E2E tests locally, follow these steps:

1. Clone this repository to your local machine: git clone https://github.com/RubenUeber/codeleap-automatedproject.git.
2. Navigate to the project directory: cd e2e-test-cypress
3. Install dependencies: npm install
4. Start the Cypress test runner: npx cypress open

</h3>
